## 2、按自己设计的表结构，插入100万订单模拟数据，测试不同方式的插入效率  

- 用自己写的原始jdbc的方法，一般10000次连接后，可用短裤资源耗尽，创建连接失败
- 用带线程池的原生jdbc方法，很慢，小时级，没有继续测试
- 用多线程的方法，20min左右
- 批量处理的方式，分批将数据准备好，然后调用批量插入的接口，可以做到几分钟
